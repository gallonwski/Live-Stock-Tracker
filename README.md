# Live Stock Tracker

Mini trading dashboard for near real-time stock monitoring using Python.

## Features
- Real-time stock price tracking
- Multi-ticker comparison
- Technical indicators:
  - Moving averages
  - RSI
  - Bollinger Bands
- Interactive Plotly charts

## Libraries used
- yfinance
- pandas
- plotly
- requests

## How to run

Install dependencies:

pip install yfinance pandas plotly requests

Then run the notebook:

Stock_price_tracker.ipynb

## Example

Interactive dashboard showing:
- multi-ticker comparison
- technical indicators
- alerts (RSI oversold / overbought)

## Notes
Yahoo Finance intraday data may have limitations for 1-minute intervals.
